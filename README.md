Реализован алгоритм топологической сортировки (алгоритм Kahn) для определения порядка загрузки зависимостей. Порядок определяется следующим образом:

1. **Зависимости загружаются перед пакетами, которые от них зависят**
2. **Корневой пакет загружается последним** (так как он зависит от всех остальных)
3. **Пакеты без зависимостей загружаются первыми**
